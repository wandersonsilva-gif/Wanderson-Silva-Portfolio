Sobre Mim

Sou desenvolvedor iniciante apaixonado por tecnologia, programação e criação de experiências interativas. Atualmente estudo na área de computação e venho construindo minha base em lógica de programação, desenvolvimento web e boas práticas de código. Tenho experiência com Python, HTML, CSS e JavaScript, além de atuar com análise de dados, automação de sistemas e criação de agentes de IA para resolver problemas do dia a dia de forma criativa. Também possuo conhecimentos em manutenção de hardware e edição de vídeos, o que me ajuda a enxergar tecnologia de forma mais completa, desde o físico até o digital. Busco constantemente aprender coisas novas, participar de projetos práticos e evoluir como desenvolvedor, sempre focado em escrever código limpo, entender o funcionamento por trás das ferramentas e entregar soluções que gerem valor para as pessoas.

InovaBank – Banco Digital Moderno

Passe o cursor para revelar os detalhes

O InovaBank é um projeto de site de banco digital com foco em design moderno, usabilidade e experiência do usuário. A plataforma apresenta um dashboard financeiro interativo, onde o usuário pode visualizar saldo disponível, variação mensal, cartões digitais, histórico de transações e gastos mensais. O layout utiliza tema escuro, elementos visuais suaves e uma interface limpa, simulando um ambiente real de internet banking. O projeto foi desenvolvido com HTML e CSS, com o objetivo de demonstrar habilidades em UI/UX, organização de layout, responsividade e criação de interfaces profissionais para aplicações financeiras.
